## Key Features

1. **Interactive Portfolio (`index.html`)**:
   - Modern glassmorphic dark design.
   - Built with pure HTML/CSS and vanilla JS (no heavy external framework dependencies).
   - **Interactive Agent Sandbox**: A simulated CLI/Shell terminal widget matching my focus on AI agentic workflows (LangGraph). Recruiter/visitor-friendly quick query commands like `about`, `skills`, `experience`, and `contact`.
   - Responsive layouts optimized for all devices.
   
2. **Traditional ATS-Friendly Resume (`resume.html`)**:
   - Clean, professional, dual-column header layout.
   - Tailored to pass modern applicant tracking systems (ATS) with a borderless table layout for Core Technical Skills.
   - Print-optimized style blocks. You can print/save as PDF directly from the page.

3. **Content Markdown Source (`resume.md`)**:
   - A markdown version of the resume keeping content fully structured and human-readable.
