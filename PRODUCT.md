# Product

## Register

brand

## Users
Recruiters, hiring managers, engineering leaders, and developers looking to hire a Senior Software Engineer specializing in AI agents and API platforms.

## Product Purpose
To showcase Shaikh Riyaz Rajjak's 11 years of technical experience, credentials, and expertise in API gateways and agentic AI systems through a polished, interactive web experience.

## Brand Personality
Technical, committed, and forward-looking.

## Anti-references
- Boring, generic, flat template sites that look like standard resume generators.
- Low-contrast light/dark designs that are hard to read.
- Non-functional animations that add visual clutter without utility.

## Design Principles
- **Showcase through Interaction**: Use functional interactive elements (like the CLI console widget) to prove technical capability.
- **Flawless Readability**: Enforce high text contrast and clear hierarchy across both "Aurora" (light) and "Solar" (dark) themes.
- **ATS and Print Ready**: Keep the main resume section clean, tabular, and print-optimized.

## Accessibility & Inclusion
- WCAG AA contrast compliance for all text.
- Clean semantic HTML structure for screen readers.
- Respect prefers-reduced-motion media queries.
